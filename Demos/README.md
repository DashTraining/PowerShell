# PowerShell Demo Scripts

This directory contains PowerShell demo scripts used for training and practical examples. Each script is described below.

| Script | Description |
|---|---|
| `1.03 ForEach-Object LAB.ps1` | Pipeline lab showing `ForEach-Object` processing, object transformation, and scriptblock usage. |
| `1.06 DateTime type.ps1` | DateTime creation, parsing, formatting, conversion, and edge-case examples. |
| `1.08 Control Flow.ps1` | Control flow examples for `if`, `switch`, loops, and decision logic. |
| `1.08 Profile Script.ps1` | Demonstrates PowerShell profile script location, customization, and startup behavior. |
| `1.09 ServiceTools SIMPLE.psm1` | Simple module example that retrieves Windows service start information. |
| `2.02 COM Excel Application.ps1` | Automates Excel using COM from PowerShell. |
| `2.02 MailKit.ps1` | Email operations using the MailKit .NET library. |
| `2.02 WebRequest and RestMethod.ps1` | Web request and REST API examples using `Invoke-WebRequest` and `Invoke-RestMethod`. |
| `2.03 Text Menus.ps1` | Text-based interactive console menu demo. |
| `2.05 String Conversion.ps1` | String parsing and conversion examples for PowerShell data handling. |
| `2.08 Workflows.ps1` | PowerShell Workflow example showing workflow structure and activities. |
| `3.00 ArgumentCompleter.ps1` | Custom tab-completion example for advanced argument handling. |
| `3.00 LanguageParser.ps1` | Parses PowerShell command tokens and demonstrates AST/Parser usage. |
| `3.01 File IO.ps1` | Demonstrates file input/output operations in PowerShell. |
| `3.01 Optimization.ps1` | Optimization techniques and performance examples for scripts. |
| `3.02 Enums and flags.ps1` | Examples showing PowerShell enums and bit-flag usage. |
| `3.03 Module Publishing.ps1` | Demonstrates module publishing and manifest creation concepts. |
| `Azure Storage Account Demo.ps1` | Demo for Azure Storage account creation and management. |
| `Azure VM Demo.ps1` | Demo for Azure VM deployment and configuration. |
| `Azure VPN Certificates Demo.ps1` | Demo covering Azure VPN certificate configuration. |
| `Inside Certificates for PowerShell-DevOps Global Summit 2021.ps1` | Certificate management script used in a conference session. |
| `Zero-to-Hero-CLASSNOTES.ps1` | Class notes and examples for the Zero-to-Hero training series. |
